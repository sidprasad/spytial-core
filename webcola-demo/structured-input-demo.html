Structured Input Graph Demo

This demo showcases the StructuredInputGraph custom element - a powerful data editor with collapsible sections, constraint validation, and multi-format export.

Instructions:

  • Add Atoms: Use the "Atoms" section in the data editor panel (right side) to add nodes
  • Create Relations: Use the "Relations" section to create edges, or Cmd/Ctrl+drag between nodes
  • Collapsible Sections: Click on any section header to collapse/expand it
  • Export Data: Use the "Export" section to reify data in the appropriate format
  • Load Specs: Enter a CnD specification below to define node/edge types and constraints

Key Features:

  • Navigable UI: Collapsible sections for better organization
  • No Emojis: Clean, professional interface without distracting icons
  • Constraint Validation: Real-time validation with UNSAT core reporting
  • Smart Export: Uses data instance's reify() method (JSON, Pyret, Alloy, etc.)
  • Enhanced Endpoint Markers: Amber square for source, red triangle for target
  • Auto-generated IDs: Type-based unique identifiers with user-provided labels
