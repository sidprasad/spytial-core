Spytial Spec Editor

Schema-driven layout-spec editing with domain awareness, theming, and a selector assistant.

Domain awareness

A real AlloyDataInstance (a binary-search-tree with sig Node and fields left, right, key) powers the type/relation dropdowns and selector completions. Type in a selector field to see them.

Selector assistant (✨)

A mock assistant turns a natural-language request into a selector after a short delay (try "children" or "leaf nodes"), and lints any selector that contains TODO.

Theming

Switch between the default light theme, the built-in dark preset, and a custom "funky" theme passed via the theme prop.
