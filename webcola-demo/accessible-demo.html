Spatial Queries as Accessible Affordances

In visual tools, dragging reveals what spatial arrangements are possible. The spatial REPL provides the same exploration non-visually: must.leftOf(Node0) answers "what must always be to the left?" — information locked behind drag interaction in existing tools.

Spatial Queries (diagram)

Query what must, can, or cannot hold spatially. These answer questions locked behind drag interaction in visual tools.

Enter a spatial query like must.rightOf(Node0) and press Enter. Click results to navigate to them.
Modalities: must — true in ALL valid layouts can — true in SOME valid layout cannot — true in NO valid layout Try: must.rightOf(Node0) can.leftOf(Node1) cannot.above(Node2) must.aligned.x(Node0) grouped(Node0) contains(G1)
