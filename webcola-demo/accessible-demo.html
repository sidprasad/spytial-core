// Run "Translate to Accessible" first, then enter a spatial query above. // // Modalities: // must — true in ALL valid layouts (universally entailed) // can — true in SOME valid layout (feasible) // cannot — true in NO valid layout (infeasible) // // Examples: // must.rightOf(Node0) — nodes that MUST be right of Node0 // can.leftOf(Node1) — nodes that CAN be left of Node1 // cannot.above(Node2) — nodes that CANNOT be above Node2 // must.aligned.x(Node0) — nodes that MUST be x-aligned with Node0 // grouped(Node0) — groups containing Node0 // grouped(Node0, Node1) — groups containing both nodes // contains(G1) — members of group G1
