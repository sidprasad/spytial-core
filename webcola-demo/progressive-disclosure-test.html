How to Test Progressive Disclosure

  1. Look for groups rendered as rectangles containing multiple nodes
  2. Find the toggle button (‚àí or +) in the top-right corner of each group
  3. Click the toggle button to collapse/expand the group
  4. When collapsed:
    • Nodes inside the group should be hidden
    • The group should show a different visual style (dashed border, higher opacity)
    • Edges connected to hidden nodes should also be hidden
    • Button changes from ‚àí to +
  5. Click again to expand the group and restore full visibility
