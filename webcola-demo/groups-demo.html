Groups Constraint Demo

This demonstrates the new groupby constraint for creating multiple groups from binary selectors.

YAML Configuration

constraints:
  - groupby:
      selector: "Person->Car"
      name: "ownership"

Example Output

Given binary selector results:

  • [Alice, Tesla], [Alice, BMW]
  • [Bob, Ford], [Bob, Honda]

Creates groups:

  • ownership[Alice]: Contains Tesla, BMW
  • ownership[Bob]: Contains Ford, Honda

Benefits

  • Multiple Groups: Creates one group per unique first element
  • Dynamic Names: Group names follow pattern baseName[key]
  • Binary Selectors: Specifically designed for relationship-based grouping
  • Clean API: Separate from unary group constraint for clarity
