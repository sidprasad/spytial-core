This demo showcases the new right-click removal functionality in input mode.

How to Test Input Mode Removal:

  • Activate Input Mode: Hold Cmd (Mac) or Ctrl (Windows/Linux)
  • Create Edges: Left-click and drag between nodes
  • Edit Edges: Left-click on existing edges to modify labels
  • Remove Edges: Right-click on existing edges to remove them
  • Remove Nodes: Right-click on nodes to remove them (and connected edges)
  • Exit Input Mode: Release Cmd/Ctrl keys
