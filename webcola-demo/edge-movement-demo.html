This demo showcases the new edge movement feature that allows you to move edges from one node to another using Shift+Click.

How to Move Edges:

  • Step 1: Hold down the Shift key - you'll see blue circles appear at edge endpoints (arrows)
  • Step 2: Click on one of the blue circles to select that edge endpoint
  • Step 3: Drag your mouse to the new target node
  • Step 4: Click on the new node to complete the move
  • Note: You can move either the source (start) or target (end) of an edge

Other Features:

  • Add Nodes: Use the structured input panel on the right to add atoms
  • Create Edges: Hold Cmd/Ctrl and drag between nodes
  • Edit Edge Labels: Hold Cmd/Ctrl and click on an edge
