What do the metrics mean?

Both are squared L2 px² summed only over elements persisting across the two frames (Penlloy §6.2). positional = 0 ⇔ every persisting node is at the same coordinate as in the prior frame. relative = 0 ⇔ every persisting edge has the same vector. spytial-core's stability implements soft positional consistency: drift is bounded by lock noise (~30k px²) and by constraint pressure when new constraints would require moving a previously stable node.
