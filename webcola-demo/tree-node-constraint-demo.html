TreeNode Constraint Demo

This demo tests the WebCola constraint validation lifecycle with a binary tree structure.

Test Scenario

This demo loads a TreeNode data structure with the following constraints:

  • orientation(selector="TreeNode->TreeNode & left", directions=["below", "left"]) - Left children should appear below and to the left
  • orientation(selector="TreeNode->TreeNode & right", directions=["below", "right"]) - Right children should appear below and to the right

Testing Instructions:

  1. Click "Load TreeNode Data" to populate the graph
  2. Click "Set Constraints" to apply orientation constraints
  3. Use the structured input panel to add/remove nodes and edges
  4. Try dragging edge endpoints to reconnect them
  5. Observe the event log to see constraint validation triggered on every change
  6. Watch for "constraints-satisfied" events when UNSAT indicators are cleared
