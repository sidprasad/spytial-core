Text Fitting in WebCola Nodes Demo

Demonstrating dynamic text sizing and expand/collapse functionality

Demo 1: Dynamic Font Sizing

Nodes automatically resize their text to fit within the available space. Short text uses larger font sizes, while longer text scales down to fit.

Demo 2: Text Wrapping and Expansion

Nodes with lots of attributes or very long labels show a collapse/expand mechanism. Click the arrow buttons on nodes to see more content.

Demo 3: Mixed Content

A realistic example with various node types, sizes, and content lengths demonstrating how the text fitting adapts to different scenarios.
