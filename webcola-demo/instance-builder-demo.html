Interactive React component for building IDataInstance objects

üéØ What is InstanceBuilder?

The InstanceBuilder is a reusable React component that provides a user interface for constructing IDataInstance objects by adding and removing atoms (nodes) and relations (edges).

Key Feature: Implementation-agnostic! Works with DotDataInstance, AlloyDataInstance, or any custom IInputDataInstance implementation.

üí° Basic Usage

import React, { useState } from 'react';
import { InstanceBuilder } from 'cnd-core';
import { DotDataInstance } from 'cnd-core/data-instance/dot';

const MyApp: React.FC = () => {
  const [instance, setInstance] = useState(
    () => new DotDataInstance('digraph G {}')
  );

  const handleInstanceChange = (newInstance: IInputDataInstance) => {
    console.log('Instance updated');
    setInstance(newInstance);
  };

  return (
    <InstanceBuilder
      instance={instance}
      onChange={handleInstanceChange}
      disabled={false}
      className="my-custom-style"
    />
  );
};

üèóÔ∏è Component Features

  • Add Atoms: Create nodes with ID, label, and type
  • Remove Atoms: Delete nodes (automatically removes related relations)
  • Add Relations: Create edges between existing atoms
  • Remove Relations: Delete specific relation tuples
  • Clear All: Reset the instance to empty state
  • Accessible: Full keyboard navigation and screen reader support

üîÑ Integration with Layout Engines

After building your data instance with InstanceBuilder, you can pass it directly to any layout engine: 

// Build the instance
<InstanceBuilder instance={myInstance} onChange={handleChange} />

// Use with layout engine
<LayoutEngine instance={myInstance} />
