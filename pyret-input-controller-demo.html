Example: Building a Binary Tree

Try building Node(5, Leaf(3), Leaf(7)) by following these steps:

  1. Add a Leaf constructor with value = 3
  2. Add another Leaf constructor with value = 7
  3. Add a Node constructor with value = 5
  4. Edit the Node's left and right fields to reference the Leaf values
  5. Set the Node as the root using the star button
  6. See the generated code: Node(5, Leaf(3), Leaf(7))

Key Features

Programming-Language Optimized

  • Constructor Dropdowns - Select from predefined Pyret data types
  • Auto ID Generation - IDs generated automatically with compact display
  • PyretExpression Support - Free-form expression input
  • Real-time Preview - Live Pyret code generation

Supported Data Types

  • List: empty, link(first, rest)
  • Tree: Leaf(value), Node(value, left, right)
  • RBTree: Leaf(value), Red(value, left, right), Black(value, left, right)
  • Option: none, some(value)
